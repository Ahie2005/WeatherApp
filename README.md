Weather Application using Python

Project Description:
Developed a desktop-based Weather Application using Python that fetches real-time weather data for any city using the OpenWeatherMap API. The application provides detailed weather information including temperature, pressure, humidity, weather description, feels-like temperature, geographic coordinates, and date-time of the location.
The project integrates API handling, JSON data processing, and GUI development to create a user-friendly interface.

**Technologies Used:**

* Python
* Requests library (API integration)
* PySimpleGUI (Graphical User Interface)
* OpenWeatherMap API
* Datetime module

**Key Features:**

* Fetches live weather data using REST API
* Displays temperature, pressure, humidity, and weather description
* Shows latitude and longitude of the searched city
* Converts UNIX timestamp to readable date and time
* Handles invalid city input and runtime exceptions
* Interactive and clean GUI interface

**Skills Demonstrated:**

* API integration
* JSON data handling
* GUI development
* Error handling and exception management
* Modular programming
